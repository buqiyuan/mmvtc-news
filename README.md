# 茂名职业技术学院官网新闻APP
- 这是一个使用jsoup爬取信息的app。
- 采用viewpager+fragment+tablayout结构，支持下拉刷新和上拉加载。
- 主要功能有登录正方教务管理系统查询个人成绩、课表等信息，还有茂职院官网首页新闻、图书馆等信息。
- 最新为v2.1版本，如果v2.1版不能正常运行，请下载v1.0.14版，下载方式：打开 Branch: master并切到tags
